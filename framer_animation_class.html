Morning Lecture

THE ANATOMY OF A TWEEN

A "Tween" (short for in-between) transitions a value from State A to State B over a strict, predetermined duration.

  • Fixed Duration: A tween strictly adheres to a set clock (e.g., exactly 0.5 seconds).
  • The Graphic Curve: The easing or pacing of this timeline is defined using a Cubic Bezier curve.
  • Math vs Reality: The X-Axis represents Time. The Y-Axis represents Animation Progress.
"Values that dip below 0 or shoot above 1 on the Y-Axis create bouncy, overflow effects."

Afternoon Lecture

FRAMER'S SECRET: SPRING PHYSICS

Unlike Tweens which restrict movements to a specific clock duration, Springs are based on physical forces.

  • No Fixed Duration: A spring animation runs until friction naturally slows it to a stop.
  • Interruptibility: Because it calculates physical momentum, springs can be reversed realistically mid-flight.
  • Parameters: Controlled by Stiffness (Tension), Damping (Friction), and Mass.
"Framer defaults to Spring animations because they echo the physical world and feel infinitely smoother for interactive UI."

Afternoon Studio

LAB: PHYSICS ENGINE

Goal: Adjust the physical mass and tension of the UI element to see how spring math naturally guides movement without a fixed duration.

Stiffness (Tension) 100
Damping (Friction) 10
Mass (Weight) 1
<motion.div
  transition={{
    type: "spring",
    stiffness: 100,
    damping: 10,
    mass: 1
  }}
/>

Notice the absence of a 'duration' variable. It will physically bounce until velocity reaches zero.

Summary

WHEN TO USE WHICH?

Feature Standard Tween (Cubic Bezier) Spring Physics
Mental Model Like plotting points on a locked timeline graph. Like stretching a rubber band with weights attached.
Duration Fixed Time (e.g. 0.3s) Dynamic (Derived from momentum)
Interruption Hard snaps or forces ugly timeline resets. Momentum is fluidly conserved mid-flight.
Best Use Cases Color crossfades, opacity changes, rigid technical loaders. Layout shifts, dragging interactions, modals, spatial movement.
